## View results in development environment

Preparation

```bash
git clone git@github.com:dexteryy/Project-WebCube.git ./a-mono-repo
```

```bash
cd ./a-mono-repo
npm run setup
```

```bash
cd ./examples/webcube-todo-app
```

Run

```bash
npm run dev
```

or

```bash
npm run deploy:local
```

or

```bash
NODE_ENV=production npm run deploy:local
```

View

* http://localhost:8012/
