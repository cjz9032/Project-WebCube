## Setup

```
cd ../../
npm run update
```

## Develeopment

```
npm run dev
```

## Running

In local env:

```
npm run deploy:local
```

or

```
npm run build
npm run serve
```

Clean cache:

```
npm run clean
```

In production env:

```
NODE_ENV=production npm run build
HOST=mydomain.com PORT=80 npm run serve
```

View

* http://localhost:8013/
